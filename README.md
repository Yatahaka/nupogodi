# Apple Catcher 🍎

Аркада: лови яблоки, уворачивайся от бомб, не упусти ни одного!

**[▶ Играть онлайн](https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/)**

## Управление

| Клавиша | Действие |
|---|---|
| ← → | Двигать корзину |
| R | Рестарт после Game Over |

## Правила

- 🍎 Поймал яблоко → **+1 очко**
- 💣 Поймал бомбу → **конец игры**
- Пропустил яблоко → **−1 жизнь** (3 жизни)
- Скорость растёт каждые 5 очков
- Рекорд сохраняется в браузере

## Запустить локально

```bash
npm install
npm run dev
```

Открыть [http://localhost:5173](http://localhost:5173)

## Деплой на GitHub Pages

1. Залить код в репозиторий на GitHub
2. Перейти в **Settings → Pages → Source → GitHub Actions**
3. Запушить в `main` — сайт соберётся и задеплоится автоматически
4. Через ~1 минуту игра будет доступна по адресу:
   `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/`
